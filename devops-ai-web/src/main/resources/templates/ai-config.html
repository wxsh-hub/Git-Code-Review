配置说明
  • AI 分类功能需要配置有效的 API 密钥才能使用

  • 温度参数建议设置在 0.1-0.3 之间以保证分类准确性
  • 分类类别包括: 新功能、功能更新、Bug修复、代码重构、文档更新、性能优化、测试相关、构建相关、CI配置、样式调整、依赖更新
