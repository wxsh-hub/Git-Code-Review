产品概述 // overview

devops-ai 是一款通用的项目发布文档自动生成工具。 它从 GitLab 仓库获取提交记录（Commits），利用 AI 对提交进行智能分类整理， 一键生成结构化的发布文档，支持 Markdown 和 HTML 两种输出格式。

快速开始 // quick start

四个步骤即可完成首次文档生成：

启动系统
运行 start.bat 或 ./start.sh，访问 http://localhost:8070
配置项目
添加 GitLab 仓库配置，选择项目和分支，测试连接
配置 AI (可选)
填写 LLM API 密钥（DeepSeek / OpenAI / Anthropic）
生成文档
选择项目和参数，点击「开始生成」，在历史页面查看
# 启动命令
# Windows start.bat # 前台运行 start.bat --daemon # 后台运行 # Linux / Mac ./start.sh start # 后台启动 ./start.sh stop # 停止服务

文档生成 // document generation

路径：侧边栏 → 文档生成 | /generate

核心参数

  • 项目配置 — 选择已配置的 GitLab 项目
  • 分支 & 版本号 — 选择目标分支和版本
  • 筛选方式 — 时间范围 或 Hash 范围
  • 作者筛选 — 指定作者（可选）
  • 输出格式 — Markdown 或 HTML
  • 文档模板 — standard / compact / detailed
  • 增量生成 — 只处理新增提交
  • AI 智能分析 — 启用 LLM 分类和分析报告

提交分类体系（12 类）

新功能 功能更新 Bug 修复 代码重构 性能优化 样式调整 文档更新 测试相关 构建相关 CI 配置 依赖更新 其他变更

分类模式

  • 规则分类（默认）— 前缀关键字匹配
  • AI 分类— LLM 语义理解，准确度更高

第三方对接 API // rest api

基础地址：https://<your-host>/api/v1

异步模式（推荐）

① 提交任务 GET /api/v1/generate?projectCode=xxx ② 轮询状态（每 2-3s） GET /api/v1/tasks/{taskId} ③ 下载结果 GET /api/v1/tasks/{taskId}/download

同步模式（CI/CD）

等待最多 120 秒直接返回文件 GET /api/v1/generate?projectCode=xxx&sync=true

错误码

200 成功 · 400 参数错误 · 404 不存在 · 408 超时 · 409 未完成 · 500 错误
Jenkins GitLab CI GitHub Actions — CI/CD 流水线原生支持
